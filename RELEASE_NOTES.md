# 초록멀티 v1.7 새 버전 안내

이번 v1.7 에서는 자주 하시는 작업이 더 빠르고 편해지도록 새 기능과 단축키가
추가되었고, 그 동안 보고된 여러 불편 사항도 함께 정리했습니다. 어떤 점이
달라졌는지 차근차근 알려드릴게요.

## ✨ 새로 추가된 기능

### 1. 게시물 목록에서 첨부파일 바로 받기 — D 키
게시물을 열지 않고 **D 키** 한 번으로 그 게시물의 첨부파일을 다운로드 폴더에
바로 저장합니다. 다운로드 진행은 비프음으로 안내되고, **Ctrl+J** 로 진행
상태를 확인할 수 있습니다. DAISY 도서 ZIP 이 감지되면 자동으로 변환 안내가
뜹니다.

### 2. DAISY 도서 변환 단축키 — Ctrl+Alt+D
어느 화면에서나 **Ctrl+Alt+D** 를 누르면 DAISY 도서 변환 대화상자가 바로
열립니다. (도구 메뉴 > DAISY 도서 변환 과 동일.)

### 3. 구독 목록 단축키 — Ctrl+Alt+L
**Ctrl+Alt+L** 을 누르면 구독 목록 대화상자가 바로 열립니다. (즐겨찾기 메뉴 >
구독 목록 보기 와 동일.)

### 4. 화면 테마 단축키 복원 — F6 / Shift+F6
화면 테마(글씨 색·배경색) 를 빠르게 바꿀 수 있는 단축키가 다시 동작합니다.
- **F6**: 다음 테마로 변경
- **Shift+F6**: 이전 테마로 변경
- 설정 메뉴에서도 같은 항목을 선택해 변경할 수 있습니다.

### 5. 동호회 관리자용 — 공지 등록 체크박스
동호회 관리자 등급으로 로그인된 경우, 게시물 작성 창의 제목 다음에
**"공지로 등록"** 체크박스가 표시됩니다. 체크 후 등록하면 해당 글이 게시판
상단에 공지로 노출됩니다. 일반 회원에게는 보이지 않습니다.

### 6. 초록등대 자료실 로그아웃 메뉴
도구 메뉴에 **"초록등대 자료실 로그아웃"** 항목이 추가되었습니다. 선택하면
확인 대화상자가 뜨고, 진행하면 마운트된 드라이브를 분리하고 저장된 자료실
아이디·비밀번호도 함께 삭제합니다. 다음 연결 때 다시 입력하면 됩니다.

### 7. 구독 알림 주기 기본값 10초
새 글 알림이 더 빨리 들어오도록 게시판 구독 알림의 기본 확인 주기를 1분 →
**10초** 로 단축했습니다. 설정(F7) 의 알림 항목에서 변경할 수 있습니다.

### 8. 명령 팔레트 → 명령 도구 모음 (이름 변경)
**Ctrl+P** 로 모든 기능을 키워드로 검색·실행하는 창의 이름이 "명령 팔레트"
에서 **"명령 도구 모음"** 으로 바뀌었습니다. 메뉴바·대화상자 제목·설명서 모두
새 이름으로 표시됩니다.

### 9. 7. 전자도서관 하위 메뉴 펼침
사이트 응답 구조상 자동 인식이 어려웠던 **7. 전자도서관** 하위 메뉴가 모두
정상으로 펼쳐지도록 보강했습니다.
- **1. 소설** 안에 일반소설 / 로맨스 / 무협·판타지 / 추리·스릴러 / 역사소설
- **2. 시·에세이** 안에 시 / 에세이
- 그 외 3.경제·경영, 4.정치·사회, 5.인문, 6.역사, 7.과학·기술·IT,
  8.건강·심리, 9.국어·외국어, 10.유아·어린이·청소년, 11.종교, 12.기타,
  55.전자도서 신청란, 77.공지사항, 88.문의, 99.전체자료실 모두 노출.

## 🛠 불편함을 정리한 부분 (버그 수정)

### 1. 7. 전자도서관 게시판 — "접근권한이 없습니다" 자동 해결
전자도서관 게시판에 들어갔을 때 **"게시판 접근권한이 없습니다.(10000)"** 이
뜨던 문제를 해결했습니다. 게시판 호출 직전 hub 페이지를 자동으로 한 번 거치고
브라우저와 동일한 헤더로 다시 시도해, 메뉴 클릭과 동일한 권한으로 접근됩니다.
혹시 거부되면 저장된 자격증명으로 자동 재로그인 후 한 번 더 시도하므로 다시
로그인할 필요가 없습니다.

### 2. Alt+G 바로가기 — hims, xvsrd 같은 코드 동작
바로가기 대화상자에 **hims, xvsrd** 같이 메뉴에 등록되지 않은 클럽 코드를
입력해도 정상으로 그 메뉴로 진입합니다. 여러 카테고리(초록등대, 동호회,
개발자 포럼, 자료실, 잡지, 전자도서관, 학습지원센터, 점자도서관) 의 hub 를
순차로 검색해 코드를 가진 항목을 찾아 클릭한 것과 동일하게 이동합니다.

### 3. Alt+G 바로가기 — 타이틀바에 정확한 이름 표시
바로가기 코드(예: green3) 로 들어갔을 때 타이틀바에 **"3. 우리들의 이야기"**
같은 정확한 메뉴 이름이 표시됩니다. (이전엔 "글쓰기", "2 페이지" 같은 엉뚱한
버튼/페이지 텍스트가 잡히던 문제 해결.)

### 4. 메인 메뉴 — "6. 소리샘 자료실" 중복 표시 해결
메인 메뉴에 같은 이름의 "6. 소리샘 자료실" 이 두 번 나오던 문제를 해결했습니다.
처음 실행 시 자동으로 한 항목으로 정리됩니다.

### 5. 자료실 / 엔터테인먼트 자료실 — 불필요 메뉴 숨김
자료실(cl=green4) / 엔터테인먼트 자료실(cl=green6) 진입 시 첫 두 항목으로
표시되던 **"1. 일반 동호회"** / **"2. 초록등대"** (한 단계 위 카테고리로
돌아가는 navigation 링크) 가 더 이상 노출되지 않습니다.

### 6. 한 글자 메뉴 이름 표시
**"시"** 같은 한 글자 메뉴 이름이 짧다는 이유로 잘려 보이지 않던 문제를
해결했습니다.

### 7. 댓글 영역 키 조작 개선
- 일부 환경에서 위/아래 방향키가 동작하지 않던 문제를 보강하기 위해
  **PgUp/PgDn** 으로도 댓글 이동이 가능하도록 추가.
- 댓글 진입 시 커서가 항상 끝에 가 있어 **Ctrl+→** 가 동작하지 않던 문제를
  해결 — 이제 커서가 시작점에 위치해 단어 단위 정방향 이동이 정상.

## 단축키 안내(Ctrl+K) 정리

단축키 안내 대화상자에 변경 이력 설명이 길게 들어가 번잡하던 부분을 정리하고,
**v1.7 신규 단축키만 깔끔하게** 보이도록 다듬었습니다.

```
Ctrl+B           즐겨찾기 열기
Ctrl+D           현재 위치를 즐겨찾기에 추가
Ctrl+P           명령 도구 모음
Ctrl+Shift+S     현재 게시판 구독 / 구독 해제
Ctrl+Alt+L       구독 목록 보기
Ctrl+Alt+D       DAISY 도서 변환
F6 / Shift+F6    다음 / 이전 화면 테마
D                게시물 목록에서 첨부파일 즉시 다운로드
Alt+1 ~ Alt+9    댓글 입력창에서 빠른 답장 템플릿 삽입
```
