## 변경 사항

- 게시판/댓글/메일 본문/쪽지 본문에서 **팝업 키·우클릭으로 상황별 컨텍스트 메뉴** 호출
- 게시물 본문 팝업 메뉴에 **이전/다음 게시물 이동** 추가
- 메일·쪽지 **실시간 알림 강화** — "즉시"(1초) 옵션 추가, 기본값 "즉시"
- 메일·쪽지 목록에 **읽음/안 읽음 상태** 표시 + 안 읽음 파서 보강
- 메일·쪽지함에서 **Shift + 좌/우 방향키 필드 순회** (게시판 목록과 동일)
- 모달 사용 중에는 새 알림 사운드·TTS·팝업 억제(작업 흐름·포커스 보호)
- 메일함 새로고침·삭제 시 엉뚱한 메일 알림이 뜨던 문제 수정
- 메일 보기 창 제목에 메일 제목 표시
- **사용자 편집 메뉴 파일** `data/menu_list.txt` 지원 + 자동 메뉴 감지
- 댓글 입력창 줄바꿈 보존 / 마침표 등 입력이 가로채지던 문제 수정
- 초록등대 자료실 자동 연결 시 모달 팝업 제거(TTS 안내만)
- **단일 인스턴스 실행** — 중복 실행 시 안내 메시지 후 종료
