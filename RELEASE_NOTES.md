# 초록멀티 v1.7.2 — 일반 동호회 메뉴 문제 해결

## 버그 수정

1. **동호회 하위 메뉴에서 "일반 동호회" 항목이 누락되던 문제 해결**

메인 메뉴의 "4. 동호회" (`/?mo=potion`) 에 들어갔을 때 "일반 동호회" (`/?mo=circle`) 항목이 표시되지 않던 문제를 해결했습니다.

원인: v1.7 에서 자료실 / 엔터테인먼트 자료실 sub-menu 의 navigation 노이즈 ("1. 일반 동호회", "2. 초록등대") 를 제거하기 위해 `/?mo=circle` 과 `/?mo=potion` URL 을 sub-menu 항목에서 무조건 거부하던 코드가, 동호회 hub 자체에 들어갔을 때도 자식 항목인 "일반 동호회" 를 함께 가렸습니다.

수정: 거부 조건을 `current_cl` 이 있을 때(클럽 내부, 예: cl=green4) 로 한정. 동호회 카테고리 (current_cl 비어 있음) 진입 시에는 "일반 동호회" 가 정당한 자식 sub-menu 로 정상 노출됩니다. 자료실 / 엔터테인먼트 자료실에서 navigation 노이즈가 가려지는 동작은 그대로 유지됩니다.
