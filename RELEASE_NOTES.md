## v1.6 변경사항 요약

### 상황별 팝업(컨텍스트) 메뉴 추가
- 메뉴 키 / Shift+F10 / 마우스 우클릭으로 화면별 작업 메뉴 호출
- 게시물 본문·댓글·메일·쪽지 본문·메일함·쪽지함·게시물 목록 모두 지원
- 게시물 본문 팝업에 이전/다음 게시물 이동 포함
- 이동·페이지 탐색 같은 내비게이션은 의도적으로 제외

### 메일·쪽지 실시간 알림 강화
- 알림 주기 "즉시"(1초) 옵션 추가, **기본값으로 적용**
- 시작 시 미확인 메일·쪽지 자동 안내
- 메일·쪽지 읽음/삭제 시 제목 표시줄 즉시 갱신
- 모달 사용 중에는 알림 사운드·TTS·팝업 억제
- 메일함 reload·삭제 시 엉뚱한 메일이 새 메일로 잘못 안내되던 문제 수정

### 메일·쪽지 목록·창 개선
- 받은함 항목 맨 앞에 **"안 읽음 / 읽음"** 상태 표시
- 메일·쪽지함에서 **Shift + 좌/우 방향키 필드 순회** (게시판과 동일)
- 메일 보기 창 제목에 메일 제목 표시
- 안 읽음 파서 보강 — 아이콘·CSS 클래스·이미지 alt 등 다양한 표시 방식 인식

### 댓글 영역
- 댓글 입력창의 줄바꿈 보존 (Enter 줄바꿈, Ctrl+Enter 등록)
- 입력 중 마침표 등이 다른 문자로 바뀌던 문제 수정
- 댓글 본문 끝에 "빈줄" 이 발화되던 문제 수정
- Home/End 키로 첫·마지막 댓글로 이동 시 전체 내용 낭독

### 사용자 편집 메뉴 + 자동 메뉴 감지
- `data/menu_list.txt` 편집으로 메뉴 구성 사용자 지정
- 사이트 메뉴 변경 시 다음 로그인 때 자동 재감지
- 설정 메뉴에 "메뉴 목록 파일 편집/다시 읽기/자동 감지로 초기화" 추가

### 카테고리 페이지 하위 메뉴
- /?mo=potion (동호회), /?mo=prg (개발자 포럼) 등 카테고리 랜딩
  페이지의 하위 메뉴가 정상적으로 표시되도록 파서 보강
- "소리샘 동사무소" 같은 공통 헤더 항목이 하위 메뉴에 끼지 않도록 필터 보강

### 프로그램 흐름·접근성
- **단일 인스턴스 실행** — 중복 실행 시 안내 후 종료
- 초록등대 자료실 자동 연결 시 모달 팝업 대신 음성 안내만
- 메일·쪽지 알림 메뉴에 "10초"·"즉시" 선택지 추가

### 자동 업데이트 안정화
- GitHub API rate limit (HTTP 403) 시 릴리스 페이지(HTML) 폴백
- 응답을 10분 로컬 캐시
- 설정 창에서 "지금 업데이트 확인" 버튼이 모달 충돌 없이 동작
- 릴리스 자산 파일명을 `greenmulti_` 접두사 + ASCII 로 통일

### 사용자 설명서·단축키 안내 갱신
- Shift+F1 사용자 설명서 대화상자 / `data/manual.txt` 에 v1.6 변경 내용 반영
- 단축키 안내 (Ctrl+K) 에 새 팝업 메뉴·필드 순회·댓글 입력창 단축키 추가
