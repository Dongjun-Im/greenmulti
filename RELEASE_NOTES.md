# 초록멀티 v1.8.1 — 더 빠르고 부드러운 사용 경험

안녕하세요, 초록등대 동호회 여러분!
v1.8 출시 이후 보내주신 피드백을 모아 v1.8.1 에서 **반응 속도** 와 **음성 안내 흐름** 을 다듬었습니다. 더 매끄럽게 사용하실 수 있을 거예요.

---

## ✨ 이번에 좋아진 점

### 1. 화살표 키 반응이 즉각 빨라졌습니다 (특히 보이스위드)
- 보이스위드를 쓰실 때 ↑/↓ 방향키로 목록을 빠르게 넘길 때마다 살짝 끊기던 현상을 없앴습니다.
- 이제는 키를 누르자마자 곧바로 다음 항목을 읽어드립니다.

### 2. 게시글 열 때 반응이 빨라졌습니다
- 게시판 목록에서 글을 골라 **Enter** 를 눌렀을 때 본문 창이 열리기까지의 지연을 줄였습니다.
- 클릭마다 디스크에 진단용 임시 파일을 쓰던 작업을 제거해서, 네트워크 드라이브에서도 본문이 바로 나타납니다.

### 3. 시작할 때 안내 음성이 또박또박 들립니다
- 처음 실행 시 음성이 다음 순서로 자연스럽게 이어지도록 정리했습니다.
  1. **"인증 중입니다. 잠시만 기다려 주세요."**
  2. (0.25초 텀)
  3. **"인증이 완료되었습니다. 초록멀티를 실행합니다."**
  4. (0.25초 텀)
  5. **"N개 메뉴를 불러왔습니다."**
- 이전엔 마지막 안내가 중간 안내를 끊어버려 답답하다는 의견이 있었는데, 이제 깔끔하게 들립니다.

### 4. 센스리더 사용 중 윈도우 기본 음성(SAPI)이 끼어드는 문제 해결
- 센스리더가 켜져 있는데도 가끔 "인증 진행 중" 같은 안내가 윈도우 기본 음성으로 들리던 문제를 해결했습니다.
- 진짜 스크린리더(NVDA / 보이스위드 / 센스리더) 가 한 번이라도 동작했다면, 그 후에는 SAPI 가 절대 끼어들지 않습니다.

### 5. 이전 게시물 단축키를 **Alt+P** 로 통일했습니다
- 본문에서 **P 키** 한 글자로 이전 글을 보던 기능과 결을 맞춰, **Alt+P** 도 같은 동작을 합니다.
- (기존 Alt+B 는 더 이상 사용하지 않습니다 — 단축키 안내 다이얼로그와 사용자 매뉴얼도 함께 갱신.)
- 다음 게시물은 그대로 **N** 또는 **Alt+N** 입니다.

- 인증 음성과 메뉴 로드 음성이 겹쳐 들리는 문제 정리.

## 📥 설치/업그레이드

- 도움말 메뉴의 **"지금 업데이트 확인"** 또는 새 인스톨러 실행만으로 자동 업그레이드됩니다.
- 개인 설정(메뉴 편집, 저장된 로그인, 즐겨찾기, 테마/글꼴) 은 그대로 보존됩니다.
- 처음 설치하시는 분은 [GitHub 릴리스 페이지](https://github.com/Dongjun-Im/greenmulti/releases/latest) 에서 `greenmulti_v1.8_setup.exe` 를 받아 실행해 주세요.

## 🙏 v1.8 에서 들어간 큰 변경 (다시 한 번 정리)

v1.8 에서 처음 추가된 굵직한 변경도 함께 다시 안내드립니다.

- **게시판 목록·본문에 조회수 표시**, 첨부파일에 다운로드 횟수 표시.
- **N / P 키** 로 게시글 본문에서 이전·다음 글 넘기기 (이번 v1.8.1 에서 **Alt+P** 도 추가).
- **NVDA / 보이스위드 / 센스리더** 어떤 것을 쓰셔도 자동 인식.
- **로그인 비밀번호 칸** 의 마스크 발음을 모든 스크린리더에서 "별표" 로 통일.
- 로그인 다이얼로그 **Tab / Shift+Tab** 즉시 안내.
- **공지 게시물에서 이전·다음 글 이동** 시 엉뚱한 글로 점프하던 문제 해결.
