# 초록멀티 v1.8 — 더 쉽게, 더 편하게 안내드립니다

안녕하세요, 초록등대 동호회 여러분!
초록멀티 v1.8 이 공개되었습니다. 사용자분들이 자주 알려주신 불편을 모아 한꺼번에 손보고, 새 기능도 함께 담았습니다. 천천히 읽어 보시고 편하게 사용해 주세요.

---

## 🆕 이번 버전에 새로 추가된 기능

### 1. 게시판에서 조회수가 보입니다
- 게시판 목록에서 **각 글이 몇 번 읽혔는지** 댓글 수와 함께 표시됩니다.
- 게시글을 열어도 위쪽에 "조회수: 12" 처럼 보입니다.
- 첨부 파일 옆에는 "(다운로드 5회)" 처럼 몇 명이 받았는지도 표시됩니다.
- 게시판 목록에서 **Shift + 오른쪽 방향키** 를 누르면
  번호 → 제목 → 작성자 → 날짜 → 댓글 수 → **조회수** 순서로 읽어줍니다.

### 2. 게시글 본문에서 글 넘기기가 더 쉬워졌습니다
- **N 키**: 다음 글 보기
- **P 키**: 이전 글 보기
- 기존 단축키(Alt+N, Alt+B) 도 그대로 쓸 수 있습니다.
- 처음 글이나 마지막 글에 도달하면 "첫 게시물입니다." / "마지막 게시물입니다." 라고 알려드리고, 엉뚱한 글로 넘어가지 않습니다.

### 3. 어떤 스크린리더를 쓰셔도 잘 동작합니다
- **NVDA / 보이스위드 / 센스리더** 모두 자동으로 인식해서 음성 안내를 들려드립니다.
- 어떤 스크린리더를 쓰시든, 별도 설정 없이 바로 사용하실 수 있습니다.
- 초록멀티 실행 후에 스크린리더를 켜셔도 인식됩니다 (이전에는 미리 켜 두셔야 했어요).

### 4. 초록등대 인증 패키지가 새로워졌습니다
- 로그인할 때 "인증 중입니다" / "초록멀티를 실행합니다" 같은 안내 음성이 끝까지 들립니다.
- 로그인 진행 중에는 비프음으로 진행 상황을 알려드립니다.
- 다른 초록등대 동호회 프로그램(예: 동호회 회원관리 프로그램) 도 함께 업데이트되어 같은 환경을 제공합니다.

## 🐞 이번 버전에서 고쳐진 문제들

### 게시판/게시글

- **공지 게시물에서 이전·다음 글로 넘기면 엉뚱한 글이 나오던 문제** 가 해결되었습니다. 이제 게시판 목록에 보이는 순서 그대로 한 칸씩 이동합니다.
- **글 제목에 "v1.7" 처럼 숫자가 들어 있으면 "v.7" 처럼 깨져 보이던 문제** 가 해결되었습니다.
- 게시글 안에서 P 키나 Alt+B 를 눌렀을 때 안내 음성이 중간에 끊기던 현상도 해결되었습니다.
- 메인 메뉴 개수 안내가 실제 보이는 메뉴 개수(예: 16개)와 일치하도록 수정되었습니다.

### 로그인·인증

- **로그인할 때 "인증 중입니다" / "초록멀티를 실행합니다" 안내가 들리지 않던 문제** 가 해결되었습니다.
- **비밀번호 칸이 "괄호닫고" 또는 "짭짭" 같이 이상한 발음으로 읽히던 문제** 가 해결되었습니다. 이제는 모든 스크린리더에서 **"별표"** 로 또박또박 읽어드립니다.
- **Tab 또는 Shift+Tab 으로 입력 칸을 이동할 때 두 번 눌러야 읽히던 문제** 가 해결되었습니다. 이제 한 번만 눌러도 바로 "소리샘 아이디 편집창", "비밀번호 편집창" 처럼 읽어줍니다.
- 로그인 창이 처음 열릴 때 "초록등대 인증" 대화상자 제목이 끝까지 안내됩니다.
- 체크박스 안내 문구를 **"아이디/비밀번호 저장 체크상자"** 로 통일했습니다.
- NAS(자료실) 로그인 창에서도 같은 비밀번호 안내 개선이 적용되었습니다.
- NAS 로그아웃 안내 문구도 **"아이디/비밀번호"** 로 통일했습니다.

### 스크린리더 호환성

- **센스리더 / NVDA / 보이스위드 사용자분들 중 음성 안내가 SAPI(윈도우 기본 음성)로 들리던 문제** 가 해결되었습니다.
- 초록멀티를 켠 다음에 스크린리더를 실행해도 잘 인식됩니다.
