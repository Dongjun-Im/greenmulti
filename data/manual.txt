초록멀티 v1.5 사용설명서
========================
빌드 날짜: 2026-04-23

1. 프로그램 소개
----------------
초록멀티는 소리샘 시각장애인 사이트(sorisem.net)를
스크린리더 환경에서 편리하게 이용할 수 있도록 만든 데스크톱 프로그램입니다.
초록등대 동호회 회원만 사용할 수 있습니다.

제작자: 임동준
이메일: d.june0503@gmail.com
관리자 이메일: greenlightsori@gmail.com


2. 설치 및 실행
----------------
- 설치 파일("초록멀티_v1.3_setup.exe")을 실행해 설치하거나,
  포터블 폴더의 "초록멀티 v1.3.exe"를 바로 실행합니다.
- 처음 실행 시 소리샘 아이디와 비밀번호를 입력합니다.
- "아이디와 비밀번호 저장"을 체크하면 다음 실행 시 자동 로그인됩니다.
- 초록등대 동호회 회원만 인증이 가능합니다.
- 초록등대 자료실(NAS) 기능을 사용하려면 WinFSP가 이 PC에 미리
  설치되어 있어야 합니다. (설치 안 돼 있으면 앱에서 안내 팝업이 뜸)

[인증 가능 등급]
- 등급 6: 일반 회원
- 등급 7: 우수 회원
- 등급 8: 최우수 회원
- 등급 9: 명예 회원
- 동호회관리자
위 등급 중 하나여야 프로그램이 실행됩니다.
그 외 등급이거나 비회원이면 프로그램이 종료됩니다.


3. 메뉴 구조
--------------
프로그램 실행 후 메인 메뉴가 표시됩니다.
위/아래 방향키로 메뉴를 이동하고, Enter로 선택합니다.

주요 메뉴:
- 초록등대 동호회
- 소리샘 공지사항
- 개발자 포럼
- 동호회
- 잡지
- 자료실
- 전자도서관
- 노원시각장애인학습지원센터
- 점자도서관
- 설문조사
- 소리샘 동사무소
- 넓은마을 바로가기 (외부 사이트)
- 행복을 들려주는 도서관 바로가기 (외부 사이트)
- 미디어접근센터(MAC) 바로가기 (외부 사이트)
- 주요 사이트 바로가기
- LG전자 유튜브 채널


4. 단축키 안내
----------------

[메뉴 탐색]
- 위/아래 방향키: 항목 이동
- 좌/우 방향키: 글자 단위 읽기
- Ctrl+좌/우: 단어 단위 읽기
- Shift+좌/우: 필드 단위 읽기 (번호/제목/작성자/날짜)
- Enter: 항목 선택/진입
- Backspace 또는 ESC: 뒤로 가기
- Home: 첫 항목으로 이동
- End: 마지막 항목으로 이동

[이동]
- Alt+Home: 메인 메뉴로 이동
- Alt+G: 바로가기
- Ctrl+G: 페이지 이동
- PageDown: 다음 페이지
- PageUp: 이전 페이지
- Ctrl+F: 게시물 검색

[게시물]
- W: 게시물 작성
- Alt+M: 게시물 수정
- Alt+D 또는 Delete: 게시물 삭제
- Alt+R: 게시물 답변

[게시물 본문]
- B: 본문 txt 저장
- Alt+S: 첨부파일 저장
- Alt+M: 게시물 수정
- Alt+D: 게시물 삭제
- Alt+R: 게시물 답변
- C: 댓글 작성
- D: 댓글 삭제 (댓글 목록에서)
- M: 댓글 수정 (댓글 목록에서)
- N: 댓글 정렬 순서 변경
- Alt+B: 이전 게시물
- Alt+N: 다음 게시물

[기타]
- Ctrl+J: 다운로드 상태 확인
- Ctrl+K: 단축키 안내
- Ctrl+L: 로그아웃
- Ctrl+N: 초록등대 자료실(NAS) 연결
- Ctrl+M: 쪽지함 열기 (받은함/보낸함/작성)
- Ctrl+Shift+M: 쪽지 바로 쓰기
- Ctrl+Shift+E: 메일함 열기 (받은함/보낸함/작성)
- Ctrl+Shift+N: 알림 센터 열기 (새 쪽지·메일 모아 보기)
- Alt+E: 관리자에게 메일 보내기
- F1: 프로그램 정보
- Shift+F1: 사용자 설명서
- Alt+U: 업데이트 확인
- Alt+F4: 프로그램 종료

[쪽지함·메일함 내부 단축키]
- ↑/↓: 항목 이동
- Enter: 선택한 쪽지/메일 열기
- D 또는 Delete: 선택 항목 삭제
- Shift+Delete: 현재 함(받은/보낸) 전체 비우기
- R: 답장 (받은함에서만)
- N: 새 쪽지/메일 작성
- F: 새로고침
- PageDown/PageUp: 다음 페이지 내용을 목록에 더 불러오기 (누적)
- Alt+R/Alt+S: 받은함/보낸함 전환
- Alt+A: "모든 쪽지/메일 삭제" 버튼

[쪽지 보기 창]
- PageUp/PageDown 또는 Alt+P/Alt+N: 이전/다음 쪽지
- R: 답장   D 또는 Delete: 삭제   Esc: 닫기

[메일 보기 창]
- B: 본문을 텍스트 파일로 저장
- Alt+S: 첨부파일 선택해서 저장
- Alt+Shift+S: 모든 첨부파일 한 번에 저장
- PageUp/PageDown 또는 Alt+P/Alt+N: 이전/다음 메일
- R: 답장   D 또는 Delete: 삭제   Esc: 닫기

[알림 센터 (Ctrl+Shift+N)]
- Enter: 선택한 항목 열기
- D 또는 Delete: 선택한 알림 하나 지우기
- A: 모든 알림 지우기
- F: 알림 새로고침

[설정·화면 보기 (저시력 지원)]
- F7: 설정 창 열기 (테마·글꼴 크기·사운드를 한 화면에서 통합 관리)
- F6: 다음 테마로 변경
- Shift+F6: 이전 테마로 변경
- Ctrl++ (또는 Ctrl+=): 글꼴 크게 (확대)
- Ctrl+-: 글꼴 작게 (축소)
- Ctrl+0: 글꼴 크기 원래대로 (기본 12)
* 글꼴 크기는 9~40 범위로 조절할 수 있으며, 설정값은
  다음 실행 시에도 유지됩니다.


5. 주요 기능 설명
------------------

[게시물 읽기]
- 게시판에 진입하면 게시물 목록이 표시됩니다.
- 위/아래 방향키로 게시물을 이동하고, Enter로 본문을 엽니다.
- Shift+좌/우 방향키로 번호, 제목, 작성자, 날짜를 순서대로 읽을 수 있습니다.
- PageDown/PageUp으로 다음/이전 페이지로 이동합니다.

[게시물 작성/수정/삭제]
- W키를 눌러 새 게시물을 작성합니다.
- Alt+M으로 자신의 게시물을 수정합니다.
- Alt+D 또는 Delete키로 자신의 게시물을 삭제합니다.
- Alt+R로 답변 게시물을 작성합니다.

[댓글]
- 게시물 본문에서 C키를 눌러 댓글을 작성합니다.
- 댓글 목록에서 D키로 자신의 댓글을 삭제합니다.
- 댓글 목록에서 M키로 자신의 댓글을 수정합니다.
- N키로 댓글 정렬 순서(최신순/오래된순)를 변경합니다.

[첨부파일 다운로드]
- 게시물 본문에서 Alt+S를 누르면 첨부파일을 저장합니다.
- 다운로드 진행 상황이 비프음으로 안내됩니다.
- Ctrl+J로 다운로드 상태를 확인할 수 있습니다.

[검색]
- Ctrl+F로 게시물을 검색합니다.
- 검색 유형(제목, 내용, 제목+내용, 글쓴이)을 선택할 수 있습니다.

[바탕화면 바로가기]
- 메뉴에서 "바탕화면에 바로가기 만들기"를 선택하면
  바탕화면에 프로그램 바로가기가 생성됩니다.

[다운로드 폴더 설정]
- 메뉴에서 "다운로드 폴더 변경"을 선택하여
  첨부파일이 저장되는 폴더를 변경할 수 있습니다.


6. v1.1 변경사항
-----------------

[인증 방식 개선]
- 일반 회원도 정상 인증되도록 회원 확인 로직 개선
  : 기존에는 관리자 페이지(admin.member.php) 한 곳만
    조회해서 일반 회원이 인증에 실패하던 문제가 있었습니다.
  : 여러 회원 페이지 URL을 순차 조회하여
    일반 회원도 자신을 찾을 수 있도록 수정했습니다.
- 등급 기반 인증 추가
  : 다음 등급만 프로그램 실행이 허용됩니다.
    · 등급 6: 일반 회원
    · 등급 7: 우수 회원
    · 등급 8: 최우수 회원
    · 등급 9: 명예 회원
    · 동호회관리자

[메뉴 표시 개선]
- 하위 메뉴에서 누락되던 항목(자료실 등) 수정
  : 메인 메뉴와 이름이 같은 하위 메뉴 항목이
    표시되지 않던 문제를 해결했습니다.
- 하위 메뉴에서 경로 안내(브레드크럼) 링크 제거
  : 메뉴 탐색 시 불필요한 경로 안내 항목이
    목록에 나타나지 않도록 개선했습니다.

[음성 안내 개선]
- 인증 시작 시 "인증 중입니다. 잠시만 기다려 주세요."
  음성이 즉시 출력되도록 수정했습니다.

[저시력 UI 테마 추가]
- 저시력자 선호 고대비 색상 적용
  : 어두운 남색 배경(#1a1f30) + 연한 노란 글자(#fff176)
  : WCAG AAA 기준(7:1 이상) 고대비
  : 순수 검정/노랑보다 눈부심이 적어 장시간 사용에 편함
- 메뉴바, 상태바, 본문 영역을 서로 다른 음영으로 구분
- 글꼴: 맑은 고딕 12pt 굵게 (기본)
- 키보드 단축키로 실시간 글꼴 확대/축소
  · Ctrl++ : 크게 (최대 40)
  · Ctrl+- : 작게 (최소 9)
  · Ctrl+0 : 원래대로 (12)
- 글꼴 크기 설정은 자동 저장되어 다음 실행 시 유지

[아이콘]
- 초록 나무 아이콘 적용
- 바탕화면 바로가기 생성 시 초록 나무 아이콘으로 표시


7. v1.2 새로 추가/바뀐 내용 (2026-04-18)
-----------------------------------------

[화면 색깔 설정 (8가지)]
- 메뉴 "설정(S)" > "화면 테마 선택(T)" 또는 F7 키를 누르면
  원하는 색 조합을 고를 수 있어요.
- 고를 수 있는 색 조합 8가지:
    1. 하얀 바탕 + 검정 글씨 (기본값)
    2. 검정 바탕 + 노란 글씨
    3. 검정 바탕 + 하얀 글씨
    4. 검정 바탕 + 초록 글씨
    5. 남색 바탕 + 노란 글씨
    6. 파란 바탕 + 하얀 글씨
    7. 아이보리 바탕 + 남색 글씨
    8. 노란 바탕 + 검정 글씨
- F6: 다음 색으로 바꾸기
- Shift+F6: 이전 색으로 바꾸기
- 선택한 색은 자동으로 저장돼서, 다음에 실행해도 그대로 유지됩니다.

[글씨 크기 바꾸기]
- Ctrl++ : 글씨 크게 (최대 40)
- Ctrl+- : 글씨 작게 (최소 9)
- Ctrl+0 : 원래 크기(12)로 되돌리기

[바로가기 기능 개선]
- 바로가기 창(Alt+G)에서 메뉴 번호나 이름을 검색할 수 있어요.
- 각 메뉴 옆에 "바로가기 코드"가 함께 표시됩니다.
  예: "6. 자료실 (바로가기 코드: pds)"
- 메뉴에 없는 코드(예: hims)를 입력하고 엔터를 누르면
  자동으로 해당 동호회/게시판으로 바로 이동합니다.
- 하위 메뉴에도 바로가기 코드가 같이 표시됩니다.

[게시물 편집 개선]
- 게시물 수정 버튼을 누르면 제목과 본문을
  한 화면에서 함께 고칠 수 있어요.
- 수정 시작할 때 "게시물 수정"이라고 음성 안내가 나옵니다.
- "본인이 작성한 글만 수정 가능" 같은 오류 메시지가
  멀쩡한 수정 페이지에서 잘못 나오던 문제를 고쳤어요.

[게시물 목록에서 댓글 수 안내]
- 게시물 목록에서 각 글 옆에 댓글 수가 표시됩니다.
  예: "제목 [댓글 3] - 작성자 (날짜)"
- 이제 댓글이 몇 개 달렸는지 바로 확인할 수 있어요.

[댓글 표시 개선]
- 내가 쓴 댓글이 댓글 목록에서 안 보이던 문제를 고쳤어요.
- 이제 내가 쓴 댓글도 목록에 잘 나타납니다.

[게시물 본문 정보 표시]
- 게시물을 열었을 때 제목, 작성자, 작성 날짜, 작성 시간을
  본문 맨 위에 함께 표시합니다.
- 작성자를 아이디가 아닌 닉네임/이름으로 표시하도록 개선.

[빈 게시판 안내]
- 게시물이 하나도 없는 게시판에 들어가면
  "게시물이 없습니다."라고 알려줍니다.
- 예전에는 엉뚱한 메뉴가 나오던 문제를 고쳤어요.

[메뉴 경로 안내 제거]
- 하위 메뉴를 볼 때 "홈 > 동호회 > ..." 같은
  경로 안내 링크가 목록에 섞여 나오던 문제를 고쳤어요.
- 이제 진짜 하위 메뉴 항목만 보입니다.

[누락된 메뉴 표시]
- 설문조사, LG전자 유튜브 채널 등 일부 메뉴의
  하위 항목이 안 보이던 문제를 고쳤어요.

[홈/엔드 키 음성 문제 해결]
- 홈 키나 엔드 키를 눌렀을 때 첫 글자("0" 또는 "1")만
  읽어주던 문제를 고쳤어요.
- 이제 전체 항목 이름을 정확하게 읽어줍니다.

[일반 회원 인증 문제 해결]
- 일반 회원이 인증 안 되던 문제를 고쳤어요.
- 이제 일반 회원, 우수 회원, 최우수 회원, 명예 회원,
  동호회관리자 모두 정상적으로 프로그램을 사용할 수 있습니다.

[URL 바로 열기]
- 게시물 본문에서 URL(https://...) 위에 커서를 두고
  Enter를 누르면 브라우저에서 해당 주소가 열려요.
- Ctrl+U를 누르면 게시물 본문과 댓글에 있는 모든 URL을
  목록으로 볼 수 있고, 원하는 URL을 선택해서 Enter로 열 수 있어요.

[기타 안정성 개선]
- 다른 사람 컴퓨터에서 실행할 때 "DLL 파일 없음" 오류가

11. v1.5 새로 추가된 기능 (2026-04-23)

[쪽지함 (Ctrl+M)]
- 도구 메뉴의 "쪽지함 열기" 또는 Ctrl+M 을 누르면 소리샘 쪽지함이 열립니다.
- 상단에 "받은 쪽지함(Alt+R)" / "보낸 쪽지함(Alt+S)" 버튼으로 함을 전환합니다.
- ↑/↓ 방향키로 쪽지 사이를 이동하면 스크린리더가 순번·상대방·날짜·요약을
  차례로 읽어줍니다.
- Enter 로 선택한 쪽지를 엽니다. 쪽지 본문 영역에는 작성 날짜 → 작성 시간 →
  보낸(받는) 사람 → 내용 순으로 통합 표시되어, 스크린리더 아래 방향키만으로
  전체 메타 정보와 본문을 순서대로 읽을 수 있습니다.
- 쪽지 보기 창에서:
  · PageUp/PageDown 또는 Alt+P/Alt+N — 이전/다음 쪽지 바로 이동
    (목록으로 돌아가지 않고 음성 안내와 함께 넘어감)
  · R 또는 "답장" 버튼 — 보낸이에게 답장 (본문이 인용 포함된 상태로 열림)
  · D 또는 Delete — 해당 쪽지 삭제 (삭제 후 같은 자리의 다음 쪽지 자동 표시)
  · Esc — 목록으로 돌아가기
- 쪽지함 목록에서도 D / Delete(선택 삭제), Shift+Delete(현재 함 전체 비우기),
  R(답장), N(새 쪽지), F(새로고침) 단축키를 사용할 수 있습니다.
- "모든 쪽지 삭제(Alt+A)" 버튼으로 현재 함을 한 번에 비울 수도 있습니다.

[쪽지 쓰기 (Ctrl+Shift+M)]
- 도구 메뉴의 "쪽지 쓰기" 또는 Ctrl+Shift+M 을 누르면 쪽지 작성창이 바로 열립니다.

- Ctrl+S 로 바로 전송할 수 있습니다.

- 도구 메뉴의 "메일함 열기" 또는 Ctrl+Shift+E 를 누르면 소리샘 사이트 내 메일함이
  열립니다. 쪽지함과 동일한 조작 체계를 따릅니다.
- "받은 메일함(Alt+R)" / "보낸 메일함(Alt+S)" / "새 메일(Alt+N)" /

- 선택 삭제: D 또는 Delete
- 전체 비우기: Shift+Delete 또는 "모든 메일 삭제" 버튼
- 답장: R (받은함에서만)
