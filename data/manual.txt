초록멀티 v1.4 사용설명서
========================
빌드 날짜: 2026-04-23

1. 프로그램 소개
----------------
초록멀티는 소리샘 시각장애인 사이트(sorisem.net)를
스크린리더 환경에서 편리하게 이용할 수 있도록 만든 데스크톱 프로그램입니다.
초록등대 동호회 회원만 사용할 수 있습니다.

제작자: 임동준
이메일: d.june0503@gmail.com
관리자 이메일: greenlightsori@gmail.com


2. 설치 및 실행
----------------
- 설치 파일("초록멀티_v1.3_setup.exe")을 실행해 설치하거나,
  포터블 폴더의 "초록멀티 v1.3.exe"를 바로 실행합니다.
- 처음 실행 시 소리샘 아이디와 비밀번호를 입력합니다.
- "아이디와 비밀번호 저장"을 체크하면 다음 실행 시 자동 로그인됩니다.
- 초록등대 동호회 회원만 인증이 가능합니다.
- 초록등대 자료실(NAS) 기능을 사용하려면 WinFSP가 이 PC에 미리
  설치되어 있어야 합니다. (설치 안 돼 있으면 앱에서 안내 팝업이 뜸)

[인증 가능 등급]
- 등급 6: 일반 회원
- 등급 7: 우수 회원
- 등급 8: 최우수 회원
- 등급 9: 명예 회원
- 동호회관리자
위 등급 중 하나여야 프로그램이 실행됩니다.
그 외 등급이거나 비회원이면 프로그램이 종료됩니다.


3. 메뉴 구조
--------------
프로그램 실행 후 메인 메뉴가 표시됩니다.
위/아래 방향키로 메뉴를 이동하고, Enter로 선택합니다.

주요 메뉴:
- 초록등대 동호회
- 소리샘 공지사항
- 개발자 포럼
- 동호회
- 잡지
- 자료실
- 전자도서관
- 노원시각장애인학습지원센터
- 점자도서관
- 설문조사
- 소리샘 동사무소
- 넓은마을 바로가기 (외부 사이트)
- 행복을 들려주는 도서관 바로가기 (외부 사이트)
- 미디어접근센터(MAC) 바로가기 (외부 사이트)
- 주요 사이트 바로가기
- LG전자 유튜브 채널


4. 단축키 안내
----------------

[메뉴 탐색]
- 위/아래 방향키: 항목 이동
- 좌/우 방향키: 글자 단위 읽기
- Ctrl+좌/우: 단어 단위 읽기
- Shift+좌/우: 필드 단위 읽기 (번호/제목/작성자/날짜)
- Enter: 항목 선택/진입
- Backspace 또는 ESC: 뒤로 가기
- Home: 첫 항목으로 이동
- End: 마지막 항목으로 이동

[이동]
- Alt+Home: 메인 메뉴로 이동
- Alt+G: 바로가기
- Ctrl+G: 페이지 이동
- PageDown: 다음 페이지
- PageUp: 이전 페이지
- Ctrl+F: 게시물 검색

[게시물]
- W: 게시물 작성
- Alt+M: 게시물 수정
- Alt+D 또는 Delete: 게시물 삭제
- Alt+R: 게시물 답변

[게시물 본문]
- B: 본문 txt 저장
- Alt+S: 첨부파일 저장
- Alt+M: 게시물 수정
- Alt+D: 게시물 삭제
- Alt+R: 게시물 답변
- C: 댓글 작성
- D: 댓글 삭제 (댓글 목록에서)
- M: 댓글 수정 (댓글 목록에서)
- N: 댓글 정렬 순서 변경
- Alt+B: 이전 게시물
- Alt+N: 다음 게시물

[기타]
- Ctrl+J: 다운로드 상태 확인
- Ctrl+K: 단축키 안내
- Ctrl+L: 로그아웃
- Ctrl+N: 초록등대 자료실(NAS) 연결
- Alt+E: 관리자에게 메일 보내기
- F1: 프로그램 정보
- Alt+F4: 프로그램 종료

[설정·화면 보기 (저시력 지원)]
- F7: 설정 창 열기 (테마·글꼴 크기·사운드를 한 화면에서 통합 관리)
- F6: 다음 테마로 변경
- Shift+F6: 이전 테마로 변경
- Ctrl++ (또는 Ctrl+=): 글꼴 크게 (확대)
- Ctrl+-: 글꼴 작게 (축소)
- Ctrl+0: 글꼴 크기 원래대로 (기본 12)
* 글꼴 크기는 9~40 범위로 조절할 수 있으며, 설정값은
  다음 실행 시에도 유지됩니다.


5. 주요 기능 설명
------------------

[게시물 읽기]
- 게시판에 진입하면 게시물 목록이 표시됩니다.
- 위/아래 방향키로 게시물을 이동하고, Enter로 본문을 엽니다.
- Shift+좌/우 방향키로 번호, 제목, 작성자, 날짜를 순서대로 읽을 수 있습니다.
- PageDown/PageUp으로 다음/이전 페이지로 이동합니다.

[게시물 작성/수정/삭제]
- W키를 눌러 새 게시물을 작성합니다.
- Alt+M으로 자신의 게시물을 수정합니다.
- Alt+D 또는 Delete키로 자신의 게시물을 삭제합니다.
- Alt+R로 답변 게시물을 작성합니다.

[댓글]
- 게시물 본문에서 C키를 눌러 댓글을 작성합니다.
- 댓글 목록에서 D키로 자신의 댓글을 삭제합니다.
- 댓글 목록에서 M키로 자신의 댓글을 수정합니다.
- N키로 댓글 정렬 순서(최신순/오래된순)를 변경합니다.

[첨부파일 다운로드]
- 게시물 본문에서 Alt+S를 누르면 첨부파일을 저장합니다.
- 다운로드 진행 상황이 비프음으로 안내됩니다.
- Ctrl+J로 다운로드 상태를 확인할 수 있습니다.

[검색]
- Ctrl+F로 게시물을 검색합니다.
- 검색 유형(제목, 내용, 제목+내용, 글쓴이)을 선택할 수 있습니다.

[바탕화면 바로가기]
- 메뉴에서 "바탕화면에 바로가기 만들기"를 선택하면
  바탕화면에 프로그램 바로가기가 생성됩니다.

[다운로드 폴더 설정]
- 메뉴에서 "다운로드 폴더 변경"을 선택하여
  첨부파일이 저장되는 폴더를 변경할 수 있습니다.


6. v1.1 변경사항
-----------------

[인증 방식 개선]
- 일반 회원도 정상 인증되도록 회원 확인 로직 개선
  : 기존에는 관리자 페이지(admin.member.php) 한 곳만
    조회해서 일반 회원이 인증에 실패하던 문제가 있었습니다.
  : 여러 회원 페이지 URL을 순차 조회하여
    일반 회원도 자신을 찾을 수 있도록 수정했습니다.
- 등급 기반 인증 추가
  : 다음 등급만 프로그램 실행이 허용됩니다.
    · 등급 6: 일반 회원
    · 등급 7: 우수 회원
    · 등급 8: 최우수 회원
    · 등급 9: 명예 회원
    · 동호회관리자

[메뉴 표시 개선]
- 하위 메뉴에서 누락되던 항목(자료실 등) 수정
  : 메인 메뉴와 이름이 같은 하위 메뉴 항목이
    표시되지 않던 문제를 해결했습니다.
- 하위 메뉴에서 경로 안내(브레드크럼) 링크 제거
  : 메뉴 탐색 시 불필요한 경로 안내 항목이
    목록에 나타나지 않도록 개선했습니다.

[음성 안내 개선]
- 인증 시작 시 "인증 중입니다. 잠시만 기다려 주세요."
  음성이 즉시 출력되도록 수정했습니다.

[저시력 UI 테마 추가]
- 저시력자 선호 고대비 색상 적용
  : 어두운 남색 배경(#1a1f30) + 연한 노란 글자(#fff176)
  : WCAG AAA 기준(7:1 이상) 고대비
  : 순수 검정/노랑보다 눈부심이 적어 장시간 사용에 편함
- 메뉴바, 상태바, 본문 영역을 서로 다른 음영으로 구분
- 글꼴: 맑은 고딕 12pt 굵게 (기본)
- 키보드 단축키로 실시간 글꼴 확대/축소
  · Ctrl++ : 크게 (최대 40)
  · Ctrl+- : 작게 (최소 9)
  · Ctrl+0 : 원래대로 (12)
- 글꼴 크기 설정은 자동 저장되어 다음 실행 시 유지

[아이콘]
- 초록 나무 아이콘 적용
- 바탕화면 바로가기 생성 시 초록 나무 아이콘으로 표시


7. v1.2 새로 추가/바뀐 내용 (2026-04-18)
-----------------------------------------

[화면 색깔 설정 (8가지)]
- 메뉴 "설정(S)" > "화면 테마 선택(T)" 또는 F7 키를 누르면
  원하는 색 조합을 고를 수 있어요.
- 고를 수 있는 색 조합 8가지:
    1. 하얀 바탕 + 검정 글씨 (기본값)
    2. 검정 바탕 + 노란 글씨
    3. 검정 바탕 + 하얀 글씨
    4. 검정 바탕 + 초록 글씨
    5. 남색 바탕 + 노란 글씨
    6. 파란 바탕 + 하얀 글씨
    7. 아이보리 바탕 + 남색 글씨
    8. 노란 바탕 + 검정 글씨
- F6: 다음 색으로 바꾸기
- Shift+F6: 이전 색으로 바꾸기
- 선택한 색은 자동으로 저장돼서, 다음에 실행해도 그대로 유지됩니다.

[글씨 크기 바꾸기]
- Ctrl++ : 글씨 크게 (최대 40)
- Ctrl+- : 글씨 작게 (최소 9)
- Ctrl+0 : 원래 크기(12)로 되돌리기

[바로가기 기능 개선]
- 바로가기 창(Alt+G)에서 메뉴 번호나 이름을 검색할 수 있어요.
- 각 메뉴 옆에 "바로가기 코드"가 함께 표시됩니다.
  예: "6. 자료실 (바로가기 코드: pds)"
- 메뉴에 없는 코드(예: hims)를 입력하고 엔터를 누르면
  자동으로 해당 동호회/게시판으로 바로 이동합니다.
- 하위 메뉴에도 바로가기 코드가 같이 표시됩니다.

[게시물 편집 개선]
- 게시물 수정 버튼을 누르면 제목과 본문을
  한 화면에서 함께 고칠 수 있어요.
- 수정 시작할 때 "게시물 수정"이라고 음성 안내가 나옵니다.
- "본인이 작성한 글만 수정 가능" 같은 오류 메시지가
  멀쩡한 수정 페이지에서 잘못 나오던 문제를 고쳤어요.

[게시물 목록에서 댓글 수 안내]
- 게시물 목록에서 각 글 옆에 댓글 수가 표시됩니다.
  예: "제목 [댓글 3] - 작성자 (날짜)"
- 이제 댓글이 몇 개 달렸는지 바로 확인할 수 있어요.

[댓글 표시 개선]
- 내가 쓴 댓글이 댓글 목록에서 안 보이던 문제를 고쳤어요.
- 이제 내가 쓴 댓글도 목록에 잘 나타납니다.

[게시물 본문 정보 표시]
- 게시물을 열었을 때 제목, 작성자, 작성 날짜, 작성 시간을
  본문 맨 위에 함께 표시합니다.
- 작성자를 아이디가 아닌 닉네임/이름으로 표시하도록 개선.

[빈 게시판 안내]
- 게시물이 하나도 없는 게시판에 들어가면
  "게시물이 없습니다."라고 알려줍니다.
- 예전에는 엉뚱한 메뉴가 나오던 문제를 고쳤어요.

[메뉴 경로 안내 제거]
- 하위 메뉴를 볼 때 "홈 > 동호회 > ..." 같은
  경로 안내 링크가 목록에 섞여 나오던 문제를 고쳤어요.
- 이제 진짜 하위 메뉴 항목만 보입니다.

[누락된 메뉴 표시]
- 설문조사, LG전자 유튜브 채널 등 일부 메뉴의
  하위 항목이 안 보이던 문제를 고쳤어요.

[홈/엔드 키 음성 문제 해결]
- 홈 키나 엔드 키를 눌렀을 때 첫 글자("0" 또는 "1")만
  읽어주던 문제를 고쳤어요.
- 이제 전체 항목 이름을 정확하게 읽어줍니다.

[일반 회원 인증 문제 해결]
- 일반 회원이 인증 안 되던 문제를 고쳤어요.
- 이제 일반 회원, 우수 회원, 최우수 회원, 명예 회원,
  동호회관리자 모두 정상적으로 프로그램을 사용할 수 있습니다.

[URL 바로 열기]
- 게시물 본문에서 URL(https://...) 위에 커서를 두고
  Enter를 누르면 브라우저에서 해당 주소가 열려요.
- Ctrl+U를 누르면 게시물 본문과 댓글에 있는 모든 URL을
  목록으로 볼 수 있고, 원하는 URL을 선택해서 Enter로 열 수 있어요.

[기타 안정성 개선]
- 다른 사람 컴퓨터에서 실행할 때 "DLL 파일 없음" 오류가
  나던 문제를 고쳤어요. (필요한 파일을 같이 포함)
- 프로그램 정보(F1) 창에 빌드 날짜가 표시됩니다.


9. v1.3 새로 추가/바뀐 내용 (2026-04-22)
-----------------------------------------

[초록등대 자료실(NAS) 연결 기능 추가]
- Ctrl+N 또는 "도구" 메뉴에서 "초록등대 자료실 연결"을 선택하면
  NAS 자료실이 Windows 드라이브 문자(예: Z:)로 자동 마운트됩니다.
- 드라이브가 마운트되면 파일 탐색기에서 일반 폴더처럼
  파일을 열고, 복사하고, 저장할 수 있습니다.
- 처음 한 번 NAS 아이디/비밀번호를 입력하면 암호화 저장되어
  다음부터는 자동으로 연결됩니다.
- 잘못된 자격증명으로 연결에 실패한 경우, 앱이 원인을 안내하고
  다시 입력할 수 있도록 도와줍니다.
- 프로그램을 종료하면 드라이브는 자동으로 분리됩니다.

[NAS 연결 엔진 교체 — rclone + WinFSP 기반]
- 기존에 사용하던 Windows 내장 WebDAV 클라이언트(WebClient/MRxDAV)가
  Synology NAS 서버와 호환되지 않아 드라이브 매핑이 실패하던 문제를
  해결하기 위해, 검증된 rclone 엔진과 WinFSP 파일시스템을
  기반으로 전면 재구현했습니다.
- SSL 자체 서명 인증서 환경에서도 문제없이 연결됩니다.
- 드라이브 문자가 생성되면 실제로 폴더 읽기가 가능한지까지
  자동으로 확인한 뒤 사용자에게 연결 완료를 알립니다.
- 연결 실패 시에는 실제 오류 메시지와 함께 상세 로그 파일
  (data\rclone_mount.log)을 남겨 문제를 빠르게 진단할 수 있습니다.
- 설치 요구사항: WinFSP가 PC에 설치되어 있어야 합니다.
  설치되어 있지 않으면 앱이 다운로드 페이지를 안내합니다.

[설정 창 통합 (F7)]
- 기존에 F7 키는 "화면 테마 선택" 대화상자만 열었지만,
  이제 F7을 누르면 테마·글꼴 크기·사운드 설정을 모두 한 화면에서
  관리할 수 있는 통합 설정 창이 열립니다.
- 각 설정 변경은 즉시 반영되며 다음 실행에도 유지됩니다.

[댓글 영역 키보드 읽기 개선]
- 댓글 영역에서 좌/우 방향키를 누르면 글자 단위로 읽어줍니다.
- Ctrl+좌/우 방향키로 단어 단위로 읽을 수 있습니다.
- 스크린리더가 댓글 본문을 놓치지 않고 자연스럽게 낭독하도록
  내부 처리 방식을 조정했습니다.

[게시판 페이지 새로고침 단축키]
- 게시판 목록을 보고 있을 때 F5를 누르면 현재 페이지를
  즉시 새로고침합니다. (새 글 확인 등에 편리)

[실행 파일 표기와 설치 파일 제공]
- 실행 파일 이름이 "초록멀티.exe"에서 "초록멀티 v1.3.exe"로
  바뀌어 버전을 한눈에 알 수 있습니다.
- 실행 파일 속성(우클릭 → 속성 → 자세히)에 버전, 제작자,
  저작권 정보가 표시됩니다.
- 두 가지 형태로 배포합니다:
  1) 설치 파일: "초록멀티_v1.3_setup.exe"
     더블클릭하면 마법사를 따라 설치되고,
     시작 메뉴/바탕화면 바로가기가 자동 생성되며,
     제어판의 프로그램 목록에 등록되어 쉽게 제거할 수 있습니다.
  2) 포터블 폴더: "초록멀티 v1.3\"
     설치 없이 폴더를 복사해 바로 실행할 수 있습니다.


10. v1.4 새로 추가/바뀐 내용 (2026-04-23)
-----------------------------------------

[사용자 설명서 대화상자 추가]
- Shift+F1 키 또는 "도움말(H) > 사용자 설명서(U)" 메뉴를 선택하면
  이 설명서를 프로그램 안에서 바로 읽을 수 있는 창이 열립니다.
- 왼쪽 목록에서 ↑/↓ 방향키로 챕터를 이동하면 오른쪽에 해당 챕터
  본문이 표시됩니다.
- Enter 키를 누르면 본문 영역으로 이동해 상/하 방향키로 줄 단위로
  읽을 수 있습니다.
- 본문에서 Ctrl+PageUp / Ctrl+PageDown 으로 이전/다음 챕터로
  빠르게 이동할 수 있습니다.
- Tab 키로 챕터 목록과 본문 사이를 오갈 수 있고,
  Esc 키로 설명서 창을 닫습니다.

[검색 히스토리 기능]
- 게시판에서 Ctrl+F로 검색 창을 연 뒤 검색어 입력칸에서
  ↑/↓ 방향키를 누르면 최근 검색어 10개까지 순환해서 볼 수 있습니다.
- 히스토리에서 검색어를 선택하면 당시 사용했던 검색 유형도
  자동으로 복원됩니다.
- 검색 내역은 data/search_history.json 파일에 저장되며
  프로그램을 다시 실행해도 유지됩니다.
